## Welcome

Thank you for reviewing this portfolio. This page is designed specifically for recruiters, hiring managers, and technical screeners who want to understand quickly what this portfolio demonstrates, how it is organized, and where to look to validate specific competencies.

This is not a sample project built from a tutorial. Every document in every repository was designed to reflect the real-world work products of a senior GRC practitioner operating in a federal health environment. The system, stakeholders, risks, and controls are all fictional, but the format, methodology, and technical depth reflect 11 years of hands-on GRC practice.

## If You Are Looking For...

**FedRAMP and Federal Authorization Expertise:**
Start with [cloudvault-fedramp-ato](https://github.com/enechi-njeze/cloudvault-fedramp-ato). That repository contains the SSP, System Security Plan attachments (att-01 through att-12), Controls Implementation Summary, Security Assessment Plan, Security Assessment Report, Plan of Action and Milestones, and Change Request documentation. It demonstrates complete knowledge of the FedRAMP authorization lifecycle from categorization through ATO.

**NIST RMF Implementation:**
See [cloudvault-nist-rmf](https://github.com/enechi-njeze/cloudvault-nist-rmf). All six RMF steps are documented: Prepare (Step 1), Categorize (Step 2), Select (Step 3), Assess (Step 4), Authorize (Step 5), and Monitor (Step 6). This repository shows RMF not as a checklist but as a continuous lifecycle.

**HIPAA Security Compliance:**
See [cloudvault-hipaa](https://github.com/enechi-njeze/cloudvault-hipaa). Includes PHI inventory, gap analysis, technical safeguards implementation, Business Associate Agreement templates, workforce training program, and breach response procedure. The CHP certification validates the depth of HIPAA knowledge.

**SOX ITGC (IT General Controls):**
See [cloudvault-sox-itgc](https://github.com/enechi-njeze/cloudvault-sox-itgc). Includes Risk and Control Matrix (RCM), IAM controls documentation, change management procedures, computer operations controls, application controls, and audit evidence packages. Directly applicable to financial services and public company GRC roles.

**PCI-DSS Compliance:**
See [cloudvault-pci-dss](https://github.com/enechi-njeze/cloudvault-pci-dss). Includes CDE scope definition, network segmentation documentation, requirements implementation across all 12 PCI-DSS v4.0 requirements, scan results, and SAQ framework. Applicable to any organization that processes payment cards.

**ISO 27001 ISMS:**
See [cloudvault-iso27001](https://github.com/enechi-njeze/cloudvault-iso27001). Includes ISMS scope, risk assessment aligned to ISO 31000, Statement of Applicability for all 93 Annex A controls, risk treatment plan, internal audit program, and management review framework.

**Privacy Compliance (CCPA, GDPR-aligned):**
See [cloudvault-privacy](https://github.com/enechi-njeze/cloudvault-privacy). Includes Record of Processing Activities (ROPA), Data Protection Impact Assessments (DPIA), data subject rights workflows, Data Processing Agreement templates, breach notification procedures, and CCPA supplement.

**Vulnerability Management Program:**
See [cloudvault-vuln-mgmt](https://github.com/enechi-njeze/cloudvault-vuln-mgmt). Includes asset inventory, multi-tool scan reports, risk-based classification, remediation tracking with SLA performance, validation procedures, metrics program with KPIs, and process flow diagrams. Demonstrates FedRAMP continuous monitoring integration.

**Enterprise Risk Governance:**
See [cloudvault-risk-governance](https://github.com/enechi-njeze/cloudvault-risk-governance). Includes a scored 12-entry risk register using NIST SP 800-30 methodology, a documented scoring model with Likelihood x Impact matrix, governance workflows with monthly and quarterly review cycles, executive reporting templates, and corrective action plans with POA&M linkage.

**Cross-Framework Integration and Controls Harmonization:**
You are already in [cloudvault-master-controls](https://github.com/enechi-njeze/cloudvault-master-controls). This capstone repository demonstrates the ability to manage compliance across six frameworks simultaneously without duplicating effort, using a unified controls matrix and bilateral framework mappings.
